# Chipmunk for Java

## What is Chipmunk for Java?

Java bindings for Chipmunk physics library.

## Requirements

### For compiling

1. JDK (sun-java6-jdk, openjdk-6-jdk or gcj-4.4-jdk.)
2. C compiler (like gcc.)

### For running

1. JRE.
